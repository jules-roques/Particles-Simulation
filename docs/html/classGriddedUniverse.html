Create a GriddedUniverse. Improve efficiency while computing interractions between particles. If a particle quits the domain during the simulation, it is deleted from the gridded universe.

Paramètres
cellSidedistance with which we can neglect interractions between two particles
lowerBoundone extreme corner of the area of the universe
upperBoundthe other extreme corner, must have greater coordinates
